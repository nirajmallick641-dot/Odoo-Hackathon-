Traveloop is a travel planning website developed to help users organize and manage their trips in a simple and user friendly way. The main goal of this project is to provide an all in one platform where users can create trips, explore destinations, manage budgets, and store travel related information easily. The website is designed with a modern UI, responsive layout, and smooth navigation to improve the overall user experience.

The website includes a Dashboard section that displays important travel information such as total trips, cities visited, total budget, and upcoming trips. It also provides quick access buttons for different travel activities. Users can create personalized trips in the Create Trip section by entering details like destination city, travel dates, budget, activities, and personal notes. All created trips can be managed easily in the My Trips section.

Traveloop also contains a Travel Bookings section where users can manage flight and hotel bookings. The Explore Cities page allows users to discover popular travel destinations around the world along with estimated daily budgets and famous attractions. Users can directly add destinations to their travel plans which makes the planning process easier and more interactive.

The project also includes useful tools like Budget Planner, Packing List, and Trip Journal. The Budget Planner helps users manage travel expenses properly while the Packing List allows users to create checklists for important travel items. The Trip Journal section is useful for storing travel memories, notes, and trip related information in one place.

Additional features include Shared Plans for collaborative trip planning, a Profile section for managing personal information and travel statistics, and a Currency Converter for converting international currencies during travel planning. The overall website uses travel themed colors, card based layouts, and a clean interface to provide a modern and attractive experience.

This project was developed using HTML, CSS, and JavaScript as part of a hackathon project. The main purpose of developing Traveloop was to improve frontend development skills and create a practical real world application with multiple useful features. Working on this project also helped in learning responsive web design, UI development, navigation systems, and managing multiple sections in a single web application.
